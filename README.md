# 运动轨迹数值分析系统

## 项目简介

这是一个基于 Streamlit 的 GPX 轨迹数据分析系统，用于对运动轨迹进行数值分析和可视化。项目实现了高精度的速度计算、路程积分验证以及交互式地图展示。

## 功能特性

- 📊 **核心数据看板**: 显示总路程、时间、平均速度、最大速度等关键指标
- 🧪 **算法验证**: 对比原始 GPS 路程与积分计算路程，验证数值算法的准确性
- 🗺️ **交互式地图**: 使用 Folium 展示运动轨迹，鼠标悬停可查看详细数据
- 📈 **速度曲线**: 实时交互式速度曲线图，支持缩放和悬停查看

## 技术栈

- **前端框架**: Streamlit
- **数据处理**: Pandas, NumPy
- **地图可视化**: Folium, streamlit-folium
- **图表绘制**: Plotly

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址为 `http://localhost:8501`

## 使用方法

1. 在左侧边栏上传 GPX 文件
2. 系统会自动解析并分析轨迹数据
3. 查看核心指标、算法验证图表、地图和速度曲线

## 数值算法

项目实现了以下数值计算方法：

- **高精度速度计算**: 使用三点差分法计算瞬时速度
- **路程积分**: 通过速度积分验证 GPS 路程的准确性
- **误差分析**: 计算绝对误差和相对误差

## 注意事项

- 确保上传的 GPX 文件格式正确
- 建议使用包含时间戳的完整 GPX 文件以获得最佳分析效果
- 对于大型轨迹文件，系统会自动降采样以优化地图交互性能
